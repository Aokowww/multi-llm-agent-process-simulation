# Data

This directory contains public benchmark data used for reproducibility
checks. Only data that can be redistributed under its source license is
included.

## Contents

- `agentsimulator_loanapp/`: prepared LoanApp event log derived from the
  public AgentSimulator repository.

The AcademicCredentials data used in the primary experiment is not
redistributed here. It should be downloaded from the public artifact
associated with Chapela-Campa et al. and supplied to the scripts through
command-line arguments.
